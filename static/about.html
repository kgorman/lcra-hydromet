ABOUT THIS DASHBOARD

A live window into the Lower Colorado.

Real-time water levels, dam operations, river gauges, and rainfall for the Highland Lakes and the Lower Colorado River basin in Central Texas. Built because watching the river rise and the lakes fill — or empty — shouldn't require clicking through a maze of PDFs.

DATA SOURCE

hydromet.lcra.org

Every chart on this site is rendered from LCRA's public Hydromet system, a network of 275+ automated river and weather gauges across the basin. The dashboard polls six endpoints in parallel and re-renders every 60 seconds — no local database, no caching layer, no pre-aggregation. What you see is what LCRA published.

  • FloodStatus/GetLakeLevelsGateOps — head/tail elevation + floodgate ops
  • FloodStatus/GetNarrativeSummary — operations narrative
  • GetForecastReferences — river forecast sites
  • GetLakeLevelsForAllSites — all lake-level gauges
  • GetStageFlowForAllSites — all river stage / flow gauges
  • GetRainfallForAllSites — rainfall stations across the basin
HOW IT'S BUILT

Open source, top to bottom.

  • D3.js · all charts hand-drawn
  • FastAPI + httpx · async pass-through proxy
  • Render · deploy from render.yaml blueprint
  • No build step · single HTML + CSS + JS file, served as-is
